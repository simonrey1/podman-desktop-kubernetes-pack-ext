# Kubernetes Extension Pack for Podman Desktop

The Kubernetes Extension Pack is a collection of extensions designed to enhance the Kubernetes experience in Podman Desktop. This pack provides tools for monitoring cluster resources, managing contexts, and creating Kubernetes resources from templates.

## Included Extensions

By installing the Kubernetes Extension Pack for Podman Desktop, you'll get access to the following extensions:

### Kubernetes Dashboard

[Kubernetes Dashboard](https://github.com/podman-desktop/extension-kubernetes-dashboard) lets you monitor resources from Kubernetes clusters directly within Podman Desktop.

### Kubernetes Contexts

[Kubernetes Contexts](https://github.com/podman-desktop/extension-kubernetes-contexts) allows you to manage Kubernetes contexts, making it easy to switch between clusters and namespaces.

### Kreate

[Kreate](https://github.com/podman-desktop/extension-kreate) provides templates and documentation to create Kubernetes resources, streamlining the process of defining deployments, services, and other Kubernetes objects.

## Installation

You can install the Kubernetes Extension Pack directly inside Podman Desktop. Click on the "Extensions" tab in the sidebar and search for "Kubernetes Extension Pack".

Alternatively, you can use the container image `ghcr.io/podman-desktop/extension-pack-kubernetes:nightly` to install the nightly version of the extension pack.
